User API

register - 用户注册

POST http://47.99.216.189:8080/users/register
请求参数:
字段名 类型 是否必填 说明 示例值
username string 是 用户名，唯一标识符 "gcc"
password string 是 密码 "123456"
{ "username": "gcc", "password": "123456" }

login - 用户登录

POST http://47.99.216.189:8080/users/login
请求参数:
字段名 类型 是否必填 说明 示例值
username string 是 用户名 "gcc"
password string 是 密码 "123456"
{ "username": "gcc", "password": "123456" }

getCurrentUser - 获取当前用户信息

GET http://47.99.216.189:8080/users/me
请求参数:
字段名 类型 是否必填 说明 示例值
Authorization string 是 登录token（放在请求头中） "eyJhbGciOiJI......"
URL示例: http://47.99.216.189:8080/users/me

resetPassword - 重置密码

PUT http://47.99.216.189:8080/users/resetpwd
请求参数:
字段名 类型 是否必填 说明 示例值
oldPassword string 是 旧密码 "123456"
newPassword string 是 新密码 "654321"
confirmPassword string 是 确认新密码 "654321"
{ "oldPassword": "123456", "newPassword": "654321", "confirmPassword": "654321" }

upgradeAdmin - 升级管理员(需要管理员权限)

PUT http://47.99.216.189:8080/users/upgrade?userId=5
请求参数:
字段名 类型 是否必填 说明 示例值
userId string 是 用户id 5
URL示例: http://47.99.216.189:8080/users/upgrade?userId=5

updateUser - 更新用户信息

PUT http://47.99.216.189:8080/users/update
请求参数:
字段名 类型 是否必填 说明 示例值
username string 否 用户名 "gcc2"
phone string 否 手机号 "13812345678"
{ "username": "gcc2", "phone": "13812345678" }

Product API

getProduct - 查询产品信息

GET http://47.99.216.189:8080/products?id=5
请求参数:
字段名 类型 是否必填 说明 示例值
id number 是 产品ID，唯一标识符 5
URL传参示例: http://47.99.216.189:8080/products?id=5

getProductList - 查询产品列表

GET http://47.99.216.189:8080/products/list
请求参数:
字段名 类型 是否必填 说明 示例值
pageNum number 是 页码 1
pageSize number 是 每页大小 10
URL示例: http://47.99.216.189:8080/products/list

addProduct - 添加产品信息

POST http://47.99.216.189:8080/products
请求参数:
字段名 类型 是否必填 说明 示例值
title string 是 商品标题 "二手笔记本电脑"
description string 否 商品详细描述 "九成新，2022年购入..."
type string 是 商品类型 "二手物品"/"代取需求"
price number 是 价格 2500.00
publisherId number 是 发布者用户ID 1
{ "title": "二手笔记本电脑", "description": "九成新，2022年购入...", "type": "二手物品", "price": 2500.00, "publisherId": 1 }

updateProduct - 更新产品信息

PUT http://47.99.216.189:8080/products
请求参数:
字段名 类型 是否必填 说明 示例值
id number 是 产品ID，唯一标识符 5
title string 否 商品标题 "二手笔记本电脑"
description string 否 商品详细描述 "九成新，2022年购入..."
type string 否 商品类型 "二手物品"/"代取需求"
price number 否 价格 2500.00
{ "id": 5, "title": "二手笔记本电脑", "description": "九成新，2022年购入...", "type": "二手物品", "price": 2500.00 }

deleteProduct - 删除产品

DELETE http://47.99.216.189:8080/products?id=5
请求参数:
字段名 类型 是否必填 说明 示例值
id number 是 产品ID，唯一标识符 5
URL示例: http://47.99.216.189:8080/products?id=5

filter - 筛选产品

GET http://47.99.216.189:8080/products/filter
请求参数:
字段名 类型 是否必填 说明 示例值
type string 否 商品类型 "二手物品"/"代取需求"
minPrice number 否 最低价格 1000.00
maxPrice number 否 最高价格 5000.00
hours number 否 发布时间距离当前多少小时 24
status string 否 商品状态 "未售出"/"已售出"
sortField string 否 排序字段 "price"/"public_time"
sortDirection string 否 排序方向 "asc"/"desc"
{ "type": "二手物品", "minPrice": 1000.00, "maxPrice": 5000.00, "hours": 24, "status": "未售出", "sortField": "price", "sortDirection": "asc" }
