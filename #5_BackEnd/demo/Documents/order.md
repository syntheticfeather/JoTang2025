# Order API

# createOrder

###  创建订单

**请求方式：** POST

**网址:** http://192.168.6.128:8080/orders

**请求参数:**

| 字段名         | 类型     | 是否必填 | 说明      | 示例值              |
| ----------- | ------ | ---- | ---------------- |-|
| productId   | long   | 是    | 商品id      | 1            |
| buyerId     | long   | 是    | 买家id      | 2            |


**请求示例(请求体):**

```
{
  "productId": 1,
  "buyerId": 2
}
```

**回应结构:**

```
{

}
```
